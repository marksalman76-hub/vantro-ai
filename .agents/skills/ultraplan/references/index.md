# UltraPlan References

This directory contains reference materials for the UltraPlan skill.

## Files

### planning-patterns.md
Common implementation planning patterns, plan file structures, and phase workflow details.

### anti-patterns.md
Detailed failure modes observed in AI planning systems with examples and mitigations.

## Usage

The SKILL.md file is self-contained for most use. Reference files provide additional depth when the planning session encounters edge cases or when the user wants to customize the workflow.

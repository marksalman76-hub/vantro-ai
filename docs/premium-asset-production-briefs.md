# Premium Asset Production Briefs

## Objective

Create a premium reusable media system for:

- Landing page hero visuals
- Website/app visual previews
- UGC avatar/content generation
- Ad creative production
- Product image direction
- Product mockups
- Audio/voice packs
- Motion graphics
- White-label brand packs

## Visual Standard

The platform must look better than generic AI SaaS tools.

Required visual direction:

- cinematic
- premium
- high contrast
- glass/3D depth
- realistic product UI previews
- abstract AI team network
- no cheap emoji visuals
- no cluttered floating cards
- mobile-safe compositions
- reusable across landing, app, ads, UGC, and agent outputs

## First Asset Set

### 1. Landing Hero Loop

File targets:

- `/media/landing/hero/hero-command-centre.webm`
- `/media/landing/hero/hero-command-centre.mp4`
- `/media/landing/hero/hero-command-centre-poster.webp`

Brief:

A dark cinematic AI command centre scene with glowing agent nodes, premium SaaS dashboard panels, ecommerce/business data streams, and subtle motion. No cartoon style. No cheap robot. Abstract, premium, trustworthy.

### 2. Agent Orbit Network

File target:

- `/media/lottie/agent-orbit-network.json`

Brief:

A lightweight animated network showing 24 specialist AI agents orbiting a central command core. Smooth, premium, minimal, dark theme, blue/teal/purple glow.

### 3. Product Preview Dashboard

File target:

- `/media/landing/backgrounds/product-preview-dashboard.webp`

Brief:

A high-end dashboard preview showing AI agents, execution stream, approval queue, output quality score, active clients, and performance metrics. Must look like a real premium SaaS control centre.

### 4. UGC Avatar Pack

Folder:

- `/media/ugc/avatars`

Brief:

A reusable library of realistic creator/avatar styles across ages, ethnicities, genders, regions, and brand categories. Used for UGC creative direction and future video/avatar generation.

### 5. Voice Pack

Folder:

- `/media/audio/voice-packs`

Brief:

Reusable voice identity references for UGC, ads, AI receptionist, narration, and multilingual brand output. Store only public-safe references or approved preview files.

### 6. Ad Creative Asset Pack

Folder:

- `/media/ads/templates`

Brief:

Reusable premium ad layouts for ecommerce, SaaS, local service, coaching, consulting, beauty, fashion, food, home, and digital products.

## Rules

- Do not commit secrets.
- Do not add provider API keys to repo.
- Do not touch backend/runtime yet.
- Assets must be reusable.
- Keep file names stable because frontend will reference these slots.

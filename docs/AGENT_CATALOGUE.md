\# Agent Catalogue



\## Platform Type

Premium white-label Ecommerce AI Agent Platform.



\## Core Agent Rule

Every agent must be premium quality, client-ready, white-label compatible, and able to operate inside a multi-client tenant-isolated system.



\---



\# Agent Catalogue



\## 1. Master Orchestrator Agent

Coordinates the full ecommerce agent system.



Responsibilities:

\- Route work to the correct agent

\- Coordinate workflows

\- Monitor business health

\- Review agent outputs

\- Escalate risks to owner

\- Ensure owner approval is required for budget, spend, scaling, and strategy actions

\# Influencer Collaboration Agent



\## Core Requirement

The Ecommerce AI Agent Platform must include an Influencer Collaboration Agent that can find, score, contact, manage, and track influencers or creators that fit each ecommerce store's niche.



\---



\# Influencer Matching



\## The agent must match creators based on:

\- Store niche

\- Product category

\- Target audience

\- Country and region

\- Language

\- Accent

\- Brand style

\- Content style

\- Platform

\- Budget range

\- Audience demographics

\- Engagement quality

\- Authenticity

\- Risk level



\---



\# Creator Scoring



\## Every creator must be scored on:

\- Niche fit

\- Audience fit

\- Engagement quality

\- Content quality

\- Brand alignment

\- Authenticity

\- Region/language match

\- Previous collaboration suitability

\- Risk level

\- Estimated value



\---



\# Outreach Automation



\## The agent may automatically:

\- Build creator shortlists

\- Generate personalised outreach messages

\- Draft collaboration offers

\- Send low-risk initial outreach

\- Send follow-ups

\- Track replies

\- Track status

\- Prepare owner approval requests



\---



\# Collaboration Tracking



\## The system must track:

\- Creator name

\- Platform

\- Handle/profile URL

\- Region

\- Language

\- Audience size

\- Engagement rate

\- Niche fit score

\- Contact status

\- Reply status

\- Negotiation stage

\- Product seeding status

\- Deliverables

\- Usage rights

\- Discount codes

\- Affiliate links

\- Campaign performance

\- Notes



\---



\# Owner Approval Required



The agent must not proceed without owner approval for:

\- Paid collaborations

\- Commission agreements

\- Contracts

\- Usage-rights agreements

\- Large product seeding

\- Budget commitments

\- Any action that increases spend

\- Any action that changes campaign budget

\- Any scaling action



\---



\# Default Safety Rule

If approval is missing, unclear, expired, or failed, the collaboration action must be paused or rejected by default.



\---



\# White-Label Requirement

Client-facing influencer collaboration outputs must never expose:

\- Internal prompts

\- System instructions

\- API keys

\- Workflow logic

\- Backend provider details

\- Internal approval rules

\- Hidden scoring logic

# UGC + Audio Asset Registry

This document defines the first reusable UGC, avatar, voice, music, scene, and ad template asset registry.

## Purpose

The system needs reusable media assets for:

- UGC video generation
- realistic creator avatars
- multilingual voice generation
- ad content agents
- product image agents
- landing page videos
- brand/product motion
- white-label client delivery

## Rules

- No provider keys in the repo.
- No secrets in media files.
- Public-safe assets only.
- Paid generation requires owner approval.
- Assets should support global/localised campaigns.
- Assets should be reusable across landing page, app UI, UGC, ads, image generation, and client campaigns.

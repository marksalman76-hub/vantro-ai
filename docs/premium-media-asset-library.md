# Premium Media Asset Library

This folder stores reusable visual and audio assets for the Ecommerce AI Agent Platform.

## Purpose

The platform needs high-end reusable assets for:

- landing page hero visuals
- website/app UI previews
- UGC avatar and scene generation
- ad creative assets
- product image direction
- product mockups
- motion graphics
- Lottie animations
- Spline/3D scenes
- WebM/MP4 loops
- voice packs
- music beds

## Rules

- Do not store secrets here.
- Do not store provider API keys here.
- Do not commit raw confidential client files.
- Optimise production assets before use.
- Prefer WebP/AVIF for static images.
- Prefer WebM for lightweight landing motion.
- Use MP4 fallback when needed.
- Keep assets reusable across landing, admin, client portal, UGC, ad, image, and app agents.

## Current status

Foundation only. No backend/runtime/architecture files changed.
